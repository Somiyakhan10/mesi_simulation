# MESI Simulation Project

**Multi-Exposure Speckle Imaging (MESI) Simulation for Blood Flow Analysis**

---

## Project Overview

This project simulates **Multi-Exposure Speckle Imaging (MESI)** to analyze blood flow in tissues. MESI is a noninvasive optical imaging technique that quantifies blood flow dynamics by analyzing speckle patterns captured at multiple exposure times. This simulation aims to:

- Calculate blood flow indices from speckle contrast data.
- Visualize **blood flow distribution curves**.
- Provide a pipeline for generating and saving results for further analysis.

The project is inspired by research in **biophotonics and diffuse correlation tomography**, focusing on rodent and tissue models for blood flow and oxygenation measurements.

---

## Features

- Python-based MESI simulation.
- Visualization of blood flow curves.
- Save results as high-quality images.
- Easy-to-use pipeline for data simulation and analysis.
